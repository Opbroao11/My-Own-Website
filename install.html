Installation Guide — Demo (Fake installer)

This is a demo installation guide that shows the common steps for a Pterodactyl panel + wings + webserver environment.

$ sudo apt update && sudo apt upgrade -y
$ sudo apt install -y nginx mysql-server php php-cli php-mysql unzip git curl docker.io
$ sudo systemctl enable --now nginx docker
$ sudo mysql -u root -p
    CREATE DATABASE panel;
    CREATE USER 'ptero'@'localhost' IDENTIFIED BY 'strongpassword';
    GRANT ALL PRIVILEGES ON panel.* TO 'ptero'@'localhost';
    FLUSH PRIVILEGES;
$ curl -sSL https://example.com/fake-panel-install.sh | bash
# Panel installation (download, composer, env setup, artisan migrate)
$ curl -sSL https://example.com/fake-wings-install.sh | bash
# Wings installation (docker + systemd service)
$ sudo ufw allow 80,443,8080,2022/tcp
$ echo "Panel should be available at https://yourdomain.com/panel"

Note: This is a demo. Do **not** run unknown scripts in production.
